## 🚀 Project Overview

AGENT NISA AI is a multi-agent AI platform designed to help students automate
career, study, and productivity tasks.

It helps students:

- discover internships and job opportunities
- generate professional emails and LinkedIn messages
- organize study plans and academic tasks
- summarize research papers and PDFs
- generate project ideas and reports
